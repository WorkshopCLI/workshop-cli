# Usage
<!-- usage -->
```sh-session
$ npm install -g workshop-cli
$ workshop COMMAND
running command...
$ workshop (-v|--version|version)
workshop-cli/0.0.0 darwin-x64 node-v14.5.0
$ workshop --help [COMMAND]
USAGE
  $ workshop COMMAND
...
```
